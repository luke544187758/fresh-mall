# 微服务模块
## 用户服务模块
- Golang
- mysql 使用sqlx进行操作
- zap 日志集成
- protobuf && grpc 集成
- viper 读取本地配置，主要是nacos配置中心相关配置
- 集成nacos来获取服务相关的配置信息
- 集成consul来实现服务的注册，以及配置健康检查
